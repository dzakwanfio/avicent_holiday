# ============================================================
# REQUIREMENTS - Avicent Holiday Project
# ============================================================

## SERVER / BACKEND REQUIREMENTS
- PHP >= 8.2
- Composer (https://getcomposer.org/)
- Laravel Framework ^12.0
- Laravel Breeze ^2.4 (Authentication scaffolding)
- Laravel Sanctum ^4.0 (API token authentication)
- Laravel Tinker ^2.10.1
- Inertia.js (Laravel adapter) ^2.0
- Tightenco Ziggy ^2.0 (Route helper for JS)

## Database
- MySQL / MariaDB (recommended) OR SQLite

## FRONTEND / NODE REQUIREMENTS
- Node.js >= 18.x (https://nodejs.org/)
- NPM >= 9.x

## FRONTEND PACKAGES (via npm)
- React ^18.2.0
- React DOM ^18.2.0
- @inertiajs/react ^2.0.0
- @headlessui/react ^2.0.0
- Tailwind CSS ^3.2.1
- @tailwindcss/forms ^0.5.3
- Vite ^7.0.7
- @vitejs/plugin-react ^4.2.0
- Lucide React ^0.577.0
- Motion ^12.36.0
- Axios ^1.11.0
- Concurrently ^9.0.1

## DEVELOPMENT TOOLS
- laravel-pint ^1.24 (PHP code formatter)
- fakerphp/faker ^1.23 (Seeding dummy data)

## HOW TO INSTALL & RUN
1. Clone the repository
2. Copy .env.example to .env and configure database credentials
3. Run: composer install
4. Run: php artisan key:generate
5. Run: php artisan migrate
6. Run: npm install
7. Run: npm run dev (for development)
   OR npm run build (for production)
8. Run: php artisan serve
